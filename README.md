# Junichi Mukai — Portfolio

個人ポートフォリオサイト。Next.js 15 App Router + TypeScript + Tailwind CSS v4 で構築。
制作物として遊べるインタラクティブデモを `/works/[slug]` に埋め込み。

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` — Development server
- `npm run build` — Production build
- `npm run start` — Start the production build
- `npm run lint` — ESLint
- `npm run typecheck` — TypeScript check

## Content

- `src/content/profile.ts` — プロフィール基本情報
- `src/content/biography.ts` — 経歴
- `src/content/works.ts` — 作品リスト(ゲーム含む)
